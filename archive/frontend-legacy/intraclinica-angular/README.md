# IntraClinica Angular Legacy

Legacy Angular application snapshot kept temporarily for comparison and archival planning.

Do not treat this directory as the primary app entrypoint.

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`

## Status

- This directory is legacy.
- Active implementation should continue in [../frontend](../frontend).
- Before archival, any unique deltas still needed here must be ported into the canonical frontend.
